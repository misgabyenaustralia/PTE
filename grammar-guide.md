# 📚 Guía de Gramática Esencial – PTE Academic Reading

## Los 5 Errores Más Comunes en Fill in the Blanks

---

### ❌ Error 1: Confundir "raise" vs "rise"

| Verbo | Significado | Ejemplo |
|-------|------------|---------|
| **raise** (transitivo) | levantar / aumentar algo | "They **raised** prices." / "She **raised** concerns." |
| **rise** (intransitivo) | subir / aumentarse | "Prices **rose**." / "Temperatures are **rising**." |

**Truco:** Si hay un objeto después → RAISE. Si no hay objeto → RISE.
- "They raised ___" → RAISE (objeto: prices, concerns, awareness)
- "The temperature ___" → RISE (no objeto)

---

### ❌ Error 2: Confundir "affect" vs "effect"

| Palabra | Tipo | Ejemplo |
|---------|------|---------|
| **affect** | verbo | "Pollution **affects** health." |
| **effect** | sustantivo | "The **effect** of pollution on health is severe." |
| **effective** | adjetivo | "The treatment is **effective**." |
| **effectively** | adverbio | "It works **effectively**." |

---

### ❌ Error 3: Preposiciones con verbos de resultado

| Verbo | Preposición | Ejemplo |
|-------|------------|---------|
| result **in** | consecuencia | "Deforestation results **in** habitat loss." |
| result **from** | causa | "Habitat loss results **from** deforestation." |
| caused **by** | agente | "The damage was caused **by** flooding." |
| contribute **to** | hacia algo | "This contributes **to** climate change." |
| lead **to** | consecuencia | "This leads **to** serious problems." |

---

### ❌ Error 4: Gerundio vs. Infinitivo

**Después de estas palabras → GERUNDIO (-ing):**
- prepositions: "role **in** absorbing", "responsible **for** managing", "interested **in** studying"
- certain verbs: avoid, enjoy, consider, suggest, recommend (cuando el sujeto es el mismo)

**Después de estas palabras → INFINITIVO (to + verb):**
- "in order **to** reduce", "aim **to** achieve", "tend **to** be", "likely **to** cause"
- adjectives: "essential **to** maintain", "important **to** consider"

---

### ❌ Error 5: Collocations con "make", "do", "have", "take"

| MAKE | DO | HAVE | TAKE |
|------|----|------|------|
| make a decision | do research | have an impact | take action |
| make progress | do damage | have concerns | take part |
| make an effort | do harm | have access | take into account |
| make a suggestion | do well | have difficulty | take responsibility |
| make a contribution | do a job | have potential | take measures |

## Vocabulario Académico Esencial (AWL)

Las siguientes palabras aparecen frecuentemente en textos del PTE:

### Verbos
| Verbo | Significado | Collocation común |
|-------|------------|------------------|
| analyse | analizar | analyse data/results |
| assess | evaluar | assess the impact |
| demonstrate | demostrar | demonstrate that |
| establish | establecer | establish a link |
| identify | identificar | identify problems |
| indicate | indicar | indicate that |
| suggest | sugerir | suggest that |
| highlight | destacar | highlight the need |

### Sustantivos
| Sustantivo | Significado |
|-----------|------------|
| approach | enfoque, método |
| concept | concepto |
| evidence | evidencia |
| factor | factor |
| impact | impacto |
| policy | política (de gobierno) |
| proportion | proporción |
| sector | sector |

### Adjetivos
| Adjetivo | Significado |
|---------|------------|
| significant | significativo |
| substantial | considerable |
| widespread | generalizado, extendido |
| crucial | crucial, esencial |
| relevant | relevante |
| potential | potencial |

## Conectores para Re-order Paragraphs

### Para agregar:
- Furthermore / Moreover / In addition / Additionally

### Para contrastar:
- However / Nevertheless / Despite this / On the other hand / Yet

### Para dar ejemplos:
- For example / For instance / Such as / In particular

### Para concluir:
- Therefore / As a result / Consequently / Thus / Hence

### Para secuenciar:
- Initially / First / Subsequently / Finally / Eventually

### Para explicar:
- This means that / This is because / This is due to / This can be attributed to
