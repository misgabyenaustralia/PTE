## SECCIÓN 3: Re-order Paragraphs

### Exercise 01 — Ejercicio Físico

**Set A:** 2 → 3 → 4 → 1
> "Physical activity..." (general) → "Research shows..." (evidencia) → "For example..." (ejemplo) → "Moreover..." (punto adicional)

**Set B:** D → B → A → C → E
> D: topic sentence general → B: evidencia de estudios → A: "This is partly due to" explica por qué → C: ejemplo específico (aerobic) → E: "These findings suggest" = conclusión

**Set C:** iii → iv → i → ii → v → vi
> iii: introducción general → iv: ejemplo (OMS) → i: beneficio adicional → ii: contraste ("Despite") → v: "This is concerning" explica por qué ii es preocupante → vi: respuesta/solución

---

### Exercise 02 — Educación Digital

**Set A:** B → D → E → A → C
> B: topic (plataformas online) → D: cómo ayudan → E: "For example" → A: "However" (contraste) → C: "As a result" (conclusión)

**Set B:** B → A → D → E → C
> B: tema general → A: "This shift" refiere al crecimiento descrito en B → D: "Initially" = primer obstáculo → E: "Nevertheless" contrasta con D → C: "Consequently" = conclusión

---

### Exercise 03 — Energías Renovables

**Set A:** B → A → C → D → E
> B: topic general → A: "For instance" ejemplifica B → C: "This cost reduction" refiere a A → D: "As a result" de C → E: "However" contrasta con D

**Set B:** B → E → C → A → D → F
> B: topic general → E: lo que los científicos dicen (meta) → C: dato estadístico actual → A: "Despite this progress" contrasta con C → D: "This dependency" explica por qué A ocurre → F: "Achieving this target" refiere al target de E
